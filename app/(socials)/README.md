# Social Media Routes

This route group contains quick link pages that redirect to ChangelingVR's social media profiles.

## Available Routes

- `/discord` - Discord server
- `/instagram` - Instagram profile
- `/twitter` - X (formerly Twitter) profile; alternative route
- `/x` - X profile
- `/youtube` - YouTube channel

## Purpose

These routes provide easy-to-remember URLs for sharing and linking to ChangelingVR's social media accounts across the website.
